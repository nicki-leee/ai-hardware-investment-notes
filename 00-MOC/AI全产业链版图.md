# AI全产业链版图

## 一句话框架
AI 硬件产业链可以从 [[AI CapEx]] 开始，沿着“云厂商投资 -> AI 芯片和服务器 -> 存储与封装 -> 网络互连 -> PCB/材料 -> 电力散热 -> 设备材料”逐层拆。

```text
云厂商 / AI CapEx
  -> AI 数据中心
    -> AI服务器 / AI集群
      -> 算力芯片：GPU、ASIC、CPU、DPU/NIC
      -> 存储：HBM、DRAM、NAND、SSD
      -> 先进封装：CoWoS、2.5D、Chiplet、ABF载板
      -> 网络互连：交换芯片、网卡、光模块、光芯片、DSP
      -> PCB与材料：服务器板、交换机板、CCL、铜箔、HDI
      -> 电力散热：电源、液冷、UPS、变压器、独立发电
      -> 上游设备材料：晶圆设备、封装设备、测试设备、EDA/IP
```

## 0. 需求源头：谁在花钱

### 主要环节
- 云厂商 AI 数据中心资本开支。
- 主权 AI。
- 大模型公司训练和推理需求。
- 企业私有化 AI 部署。

### 典型公司
- 美国云厂商：Microsoft、Amazon、Google、Meta、Oracle。
- AI 平台和算力供应商：NVIDIA、AMD。
- 模型和应用生态：OpenAI、Anthropic、xAI 等。

### 大白话
这一层决定“蛋糕有多大”。如果云厂商继续加 AI CapEx，下游 GPU、HBM、光模块、PCB、电力液冷都有订单来源；如果 CapEx 下修，硬件链条会一起承压。

## 1. 算力芯片层

### 主要环节
- GPU：通用 AI 训练和推理核心。
- ASIC：云厂商或大客户定制 AI 芯片。
- CPU：主机控制和通用计算。
- DPU/NIC：网络、数据搬运、虚拟化和安全。
- 交换芯片：AI 集群网络核心。

### 典型公司
| 公司 | 位置 | 说明 |
|---|---|---|
| NVIDIA | GPU、AI 平台、网络、系统 | 当前 AI 算力生态核心，覆盖 GPU、NVLink、InfiniBand、Ethernet 交换等 |
| AMD | GPU、CPU | MI 系列 AI GPU 和 EPYC CPU |
| Intel | CPU、部分 AI 加速和代工 | 数据中心 CPU 基础盘，AI 加速器竞争力需单独观察 |
| Broadcom | ASIC、交换芯片、网络芯片 | 云厂商定制芯片和数据中心网络重要供应商 |
| Marvell | ASIC、DPU/网络、光互连相关芯片 | 重点在定制 AI 芯片、数据中心互连、网络和光电连接 |
| Arm | CPU IP | 提供 CPU 架构和 IP，不是传统意义上的芯片制造商 |

### 大白话
算力芯片是 AI 工厂里的“主发动机”。GPU 或 ASIC 越多，就越需要 HBM、先进封装、高速 PCB、光模块和电力散热。

### 投资观察点
- GPU/ASIC 出货量和平台迭代。
- 云厂商自研 ASIC 是否加速。
- AI 芯片是否继续拉动 HBM 和 CoWoS。
- 网络芯片是否随 AI 集群扩大而增值。

## 2. 存储层

### 主要环节
- HBM：高带宽内存，贴近 AI 芯片。
- DRAM/DDR：服务器主内存。
- NAND/SSD：长期数据存储和企业级存储。
- 存储控制器：管理 NAND/SSD 读写。

### 典型公司
| 公司 | 位置 | 说明 |
|---|---|---|
| SK hynix | HBM、DRAM、NAND | HBM 龙头之一，AI 存储核心公司 |
| Micron | HBM、DRAM、NAND、SSD | 美系存储龙头，受益于 AI 服务器 HBM 和数据中心存储需求 |
| Samsung Electronics | HBM、DRAM、NAND、代工 | 存储巨头，同时也有晶圆代工业务 |
| Kioxia | NAND | NAND 闪存重要供应商 |
| Western Digital / SanDisk | NAND、SSD | 数据存储相关公司 |

### Micron 在哪里
Micron 位于 **存储层**，重点包括：
- HBM：服务 AI GPU/ASIC 的高带宽内存。
- DRAM：服务器和数据中心主内存。
- NAND/SSD：数据中心长期存储。

大白话：Micron 不是做 GPU 的，而是给 AI 芯片和服务器提供“高速粮仓”和“数据仓库”。AI 越需要高带宽和大容量，Micron 的 HBM、DRAM、SSD 越重要。

### SK hynix 在哪里
SK hynix 也位于 **存储层**，其中最重要的是 HBM。

大白话：SK hynix 是 AI 芯片旁边那块高速内存的核心供应商之一。GPU 再强，如果 HBM 供不上，整套 AI 服务器也交付不了。

### 投资观察点
- HBM3E、HBM4 认证和量产。
- HBM 价格和产能。
- HBM 是否挤占普通 DRAM 产能。
- NAND 是否走向扩产和价格压力。

## 3. 晶圆制造与先进制程

### 主要环节
- 先进制程晶圆代工。
- CPU/GPU/ASIC 生产。
- 良率、产能、先进节点。

### 典型公司
| 公司 | 位置 | 说明 |
|---|---|---|
| TSMC | 晶圆代工、先进封装 | AI 芯片先进制程和 CoWoS 关键供应商 |
| Samsung Foundry | 晶圆代工 | 先进制程竞争者 |
| Intel Foundry | 晶圆代工 | 先进制程和封装布局中 |
| GlobalFoundries | 成熟制程 | 不在最核心 AI GPU 先进制程，但服务部分外围芯片 |

### 大白话
AI 芯片设计出来后，要找代工厂制造。高端 GPU/ASIC 依赖先进制程，先进制程产能和良率会影响交付。

### 投资观察点
- 3nm、2nm 等先进节点进度。
- AI 芯片产能分配。
- 代工厂资本开支。

## 4. 先进封装层

### 主要环节
- CoWoS。
- 2.5D/3D 封装。
- Chiplet。
- Interposer。
- TSV。
- ABF 载板。
- 封装测试。

### 典型公司
| 公司 | 位置 | 说明 |
|---|---|---|
| TSMC | CoWoS、先进封装 | 高端 AI GPU/ASIC 封装关键供应商 |
| ASE 日月光 | 封测 | 全球封测龙头之一 |
| Amkor | 封测 | 全球封测重要公司 |
| Ibiden | ABF 载板 | 高端封装基板供应商 |
| Unimicron 欣兴 | ABF 载板、PCB | 高端载板和 PCB |
| Shinko | 封装基板 | 高端封装基板供应商 |

### 大白话
先进封装是把 AI 芯片和 HBM 摆到一起的“超级拼装工艺”。如果封装产能不够，GPU 和 HBM 即使都有，也不一定能交付成最终产品。

### 投资观察点
- CoWoS 产能。
- HBM 和封装产能是否匹配。
- ABF 载板供需。
- 封装设备和测试需求。

## 5. 网络互连层

### 主要环节
- 数据中心交换机。
- 交换芯片。
- DPU/NIC。
- 光模块。
- 光芯片、DSP、硅光。
- 光纤、连接器。
- InfiniBand / Ethernet。

### 典型公司
| 公司 | 位置 | 说明 |
|---|---|---|
| NVIDIA | InfiniBand、Ethernet、交换机、网卡 | AI 集群网络核心玩家之一 |
| Broadcom | 交换芯片、网络芯片 | 数据中心交换芯片和网络芯片龙头 |
| Marvell | 数据中心互连、定制芯片、光电相关芯片 | AI 网络和定制硅重要公司 |
| Arista | 数据中心交换机 | 云数据中心以太网交换机重要供应商 |
| Coherent | 光器件、光通信 | 光通信上游和模块相关 |
| Lumentum | 光器件、激光器 | 光通信上游 |
| Innolight 中际旭创 | 光模块 | 中国光模块龙头之一 |
| Eoptolink 新易盛 | 光模块 | 中国光模块公司 |
| Accelink 光迅科技 | 光通信器件/模块 | 中国光通信供应商 |

### Marvell 在哪里
Marvell 主要位于 **网络互连层 + ASIC/定制芯片层**：
- 数据中心定制硅。
- 高速互连芯片。
- 光互连相关电芯片。
- 网络和存储连接相关芯片。

大白话：Marvell 不是卖整台服务器，也不是存储原厂。它更像 AI 数据中心里的“高速连接和定制芯片专家”，帮云厂商把算力芯片、网络和光互连做得更快、更省电。

### 投资观察点
- 800G、1.6T 光模块放量。
- Ethernet 是否加速替代或补充 InfiniBand。
- 交换芯片和网卡升级。
- 光模块上游 DSP、激光器、硅光是否紧缺。

## 6. PCB 与材料层

### 主要环节
- AI 服务器主板。
- GPU/ASIC 加速卡板。
- 交换机板。
- 高多层板。
- HDI。
- CCL 覆铜板。
- 低损耗材料、铜箔、玻纤布、树脂。

### 典型公司
| 公司 | 位置 | 说明 |
|---|---|---|
| Unimicron 欣兴 | PCB、载板 | 高端 PCB/载板 |
| Tripod 健鼎 | PCB | 服务器和通信相关 PCB |
| TTM Technologies | PCB | 北美 PCB 公司 |
| Ibiden | 载板、PCB | 高端封装基板 |
| 沪电股份 | PCB | A 股高速通信和服务器 PCB |
| 生益科技 | CCL、材料 | 覆铜板和电子材料 |
| 生益电子 | PCB | 服务器和通信 PCB |
| 深南电路 | PCB、封装基板 | 通信、服务器、封装基板 |
| 胜宏科技 | PCB | AI 服务器和高端 PCB 映射之一 |
| 金安国纪 / 南亚新材 | CCL | 覆铜板材料 |

### 大白话
AI 服务器和交换机里的信号速度越来越快，普通板子不够用了。PCB 和 CCL 就从“普通底板”升级成“高速公路系统”，价值量和制造难度提高。

### 投资观察点
- AI 服务器 PCB 单机价值量。
- 交换机 PCB 层数和材料升级。
- CCL 价格、认证和供需。
- 减持、扩产和价格竞争风险。

## 7. 电力、散热和数据中心基础设施

### 主要环节
- 电源。
- 变压器。
- UPS。
- 液冷。
- CDU、冷板、泵、管路。
- 机柜和整柜系统。
- 独立发电、核电、公用事业。

### 典型公司
| 公司 | 位置 | 说明 |
|---|---|---|
| Vertiv | 数据中心电力和热管理 | AI 数据中心电力散热受益公司 |
| Schneider Electric | 电气设备、能源管理 | 数据中心供配电和能效管理 |
| Eaton | 电力管理 | UPS、配电、电力管理 |
| Delta Electronics 台达电 | 电源、散热 | 数据中心电源和热管理 |
| Supermicro | AI 服务器、液冷系统 | 服务器和整机方案 |
| Quanta / Wiwynn / Foxconn | 服务器 ODM | AI 服务器整机制造 |

### 大白话
AI 数据中心不是只买 GPU。GPU 功耗越高，越需要电、变压器、UPS、液冷和机柜。电力散热是 AI CapEx 外溢的重要方向。

### 投资观察点
- 单柜功率。
- 液冷渗透率。
- 数据中心接电和电网建设。
- 电力设备交期。

## 8. 半导体设备、材料和 EDA/IP

### 主要环节
- 光刻、刻蚀、薄膜、清洗、检测量测。
- 封装设备、测试设备。
- 半导体材料。
- EDA 软件。
- IP，例如 CPU IP、SerDes IP、接口 IP。

### 典型公司
| 公司 | 位置 | 说明 |
|---|---|---|
| ASML | 光刻设备 | 先进制程关键设备 |
| Applied Materials | 半导体设备 | 薄膜、刻蚀、检测等 |
| Lam Research | 刻蚀/沉积设备 | 存储和逻辑设备 |
| KLA | 检测量测 | 良率控制关键 |
| Tokyo Electron | 半导体设备 | 多类前道设备 |
| Synopsys | EDA、IP | 芯片设计工具和 IP |
| Cadence | EDA、IP | 芯片设计工具 |
| Arm | CPU IP | CPU 架构授权 |

### 大白话
设备和 EDA 是“卖铲子”的环节。AI 芯片和 HBM 越复杂，越需要先进设备、检测、EDA 和 IP。

### 投资观察点
- 先进制程资本开支。
- HBM 和先进封装设备需求。
- 中国国产替代和出口管制。
- EDA/IP 是否受益于 ASIC 和 Chiplet。

## 10. 从投资角度怎么读这张图

### 最上游的验证
- 云厂商 CapEx 是否继续增加。
- AI 服务器出货是否上修。
- GPU/ASIC 交付周期是否延长或缩短。

### 中游瓶颈
- HBM 是否紧缺。
- CoWoS 产能是否紧缺。
- 800G/1.6T 光模块是否供不应求。
- PCB/CCL 是否有高端产能缺口。

### 下游兑现
- 公司收入是否真正增长。
- 毛利率是否提升。
- 存货和应收是否健康。
- 扩产是否过快。
