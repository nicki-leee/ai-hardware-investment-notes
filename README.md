# AI硬件投资研究笔记

这是一个面向 Obsidian 的 AI 硬件投资入门库，覆盖 AI CapEx、AI服务器、PCB、光模块、存储/HBM、半导体、先进封装，以及 AI 产业链术语词典。

## 如何使用

建议先打开：
- [[00-MOC/AI投资总览]]
- [[00-MOC/AI硬件产业链地图]]
- [[00-MOC/AI全产业链版图]]
- [[08-术语词典/AI产业链术语词典]]
- [[04-研报卡片/高质量研报清单]]

使用方式：
1. 先读 MOC，建立框架。
2. 每读一篇研报，在 `04-研报卡片` 里补摘要。
3. 每研究一家公司，复制 `02-公司/公司研究模板.md`。
4. 每周更新一次 `03-指标` 和 `06-风险清单`。

## 目录

- `00-MOC`：总览、产业链地图和完整产业链版图
- `01-产业链`：AI服务器、PCB、光模块、存储、半导体、先进封装
- `02-公司`：公司研究模板
- `03-指标`：CapEx、HBM、PCB、光模块等跟踪指标
- `04-研报卡片`：阅读路线、研报卡片和资料索引
- `05-投资假设`：投资逻辑假设
- `06-风险清单`：风险框架
- `08-术语词典`：AI 芯片、存储、先进封装、光模块、PCB、电力散热等术语解释

## 关于第三方材料

公开版不包含研报 PDF、截图或第三方原始材料，只保留原始链接、阅读卡片和个人学习笔记。请通过发布方原始页面获取报告。
