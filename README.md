# AI硬件投资研究笔记

这是一个面向 Obsidian 的 AI 硬件投资入门库，覆盖 AI CapEx、AI服务器、PCB、光模块、存储/HBM、半导体和先进封装。

## 如何使用

建议先打开：
- [[00-MOC/AI投资总览]]
- [[00-MOC/AI硬件产业链地图]]
- [[04-研报卡片/高质量研报清单]]

使用方式：
1. 先读 MOC，建立框架。
2. 每读一篇研报，在 `04-研报卡片/待读卡片` 里补摘要。
3. 每研究一家公司，复制 `02-公司/公司研究模板.md`。
4. 每周更新一次 `03-指标/每周跟踪清单.md` 和 `06-风险清单/AI硬件风险清单.md`。

## 目录

- `00-MOC`：总览和知识地图
- `01-产业链`：AI服务器、PCB、光模块、存储、半导体、先进封装
- `02-公司`：公司研究模板
- `03-指标`：CapEx、HBM、PCB、光模块等跟踪指标
- `04-研报卡片`：阅读路线、研报卡片和资料索引
- `05-投资假设`：投资逻辑假设
- `06-风险清单`：风险框架

## 关于研报 PDF

公开版不包含研报 PDF 原文件，只保留原始链接和阅读卡片。请通过发布方原始页面获取报告。
