# AMD

## 一句话定位
AMD 是 AI GPU 和数据中心 CPU 的重要供应商，是 NVIDIA 之外最受关注的 AI 加速器竞争者之一。

## 所属环节
- 产业链位置：算力芯片层。
- 对应板块：[[半导体]]、[[AI服务器]]。
- 上游：[[TSMC]]、HBM 供应商、先进封装、封测、PCB。
- 下游：云厂商、服务器 OEM/ODM、企业数据中心。
- 核心客户：云厂商和企业 AI 客户。

## AI相关业务
- 产品：Instinct AI GPU、EPYC CPU、相关加速平台。
- 收入占比：重点看 Data Center 业务。
- 产能：依赖先进制程、HBM 和封装。
- 认证进展：云厂商导入和软件生态支持。

## 投资逻辑
1. AI GPU 市场太大，第二供应商有份额提升空间。
2. EPYC CPU 在数据中心具备基础盘。
3. 如果软件生态和平台导入改善，AI GPU 收入弹性较大。

## 关键验证指标
- Data Center 收入。
- Instinct GPU 出货。
- 云厂商客户导入。
- 软件生态和开发者支持。
- 毛利率。
- 与 NVIDIA 的性能、供给和生态差距。

## 风险
- NVIDIA 生态优势强。
- AI GPU 客户导入慢。
- HBM/封装供应约束。
- 价格竞争。
- CPU 周期波动。
